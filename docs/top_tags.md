| Available | Method                                | Description
|:---------:|:-------------------------------------:| ----------------------------------------------------------------|

| NO        | **/users/{id}/top-tags** <br/> ![me](https://cdn.sstatic.net/apiv2/img/me.png?v=f1cb4f2bb0ba) **/me/top-tags** | Get the top tags (by score) a single user has posted in. |
